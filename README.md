2.  Navigate to the configuration tab to set your primary preferences.

You can maintain multiple agents at once. Use the dashboard to stop, edit, or delete any agent.

*   **Stop:** Click the red square icon to pause an active agent.
*   **Edit:** Select the gear icon to change the settings or the goal of the agent.
*   **Logs:** The bottom panel shows the history of actions for the selected agent. Use this to review what the agent finished.

## 💡 Best Practices for Success

Follow these tips to get the best results from your AI agents.

Keep your instructions simple. If an agent fails to complete a task, break the task into smaller parts. For example, tell the agent to "Find the file" first, then "Summarize the text in the file."

Name your agents based on their purpose. Names like "ResearchAssistant" or "DataSorter" help you find them quickly in the dashboard.

Refresh the agent list once a week to ensure you have the latest performance updates. The software checks for new versions of components every time you launch the application.

## ❓ Frequently Asked Questions

**Does the agent work offline?**
No. The agent requires an active internet connection to process language tasks.

**Is my data private?**
The framework processes information according to the local settings. You control which documents the agent has access to.

**Can I run two agents at the same time?**
Yes. You can run multiple instances, provided your computer has enough memory. If the system slows down, close one of the agents.

**What if the setup fails?**
Check your internet connection first. Ensure that your firewall does not block the application. If issues persist, delete the file and start the download process again.

**Where can I find more help?**
The main repository page contains a link to common issues. Browse those entries to see if other users solved your specific problem.
